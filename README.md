# SiDBaY

Repositorio inicial preparado para albergar el código de SiDBaY.

## Estructura

- `src/`: código fuente principal.
- `tests/`: suites de pruebas automatizadas y ejemplos.
- `docs/`: documentación extendida (arquitectura, decisiones, protocolos).
- `.vscode/`: configuración local de VS Code.
- `README.md`: guía rápida de instalación y uso.
- `CHANGELOG.md`: historial de cambios siguiendo SemVer.
